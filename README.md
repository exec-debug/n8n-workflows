# Public n8n Workflow Portfolio

Reusable n8n workflow templates for CRM automation, lead management, meeting operations, document intake, follow-up sequences, and business-day scheduling.

## Public-release safeguards

The templates contain no known client names, employee identities, customer records, real contact details, credential bindings, API secrets, production webhook IDs, private domains, pinned execution data, or original n8n instance metadata.

Client-specific document terminology and industry-specific sales language were generalized. All workflows are inactive. Test and draft modes are enabled where present.

## Before using a workflow

1. Import the JSON into n8n.
2. Keep it inactive.
3. Connect your own credentials.
4. Replace CRM objects, fields, workflow IDs, URLs, schedules, and message copy.
5. Test with fictional data.
6. Review every branch.
7. Activate only after validation.

## Included templates

- Active engagement and first-contact outreach
- Lead reply detection
- Dormant and stalled lead re-engagement
- Meeting detection, reminders, confirmations, and RSVP synchronization
- Internal meeting preparation
- Onboarding calendar-to-CRM synchronization
- Opportunity, proposal, post-meeting, and requested follow-up tasks
- Document intake and case processing
- US business-day and holiday calculation

## Security

Never commit real credentials, `.env` files, access tokens, customer payloads, execution exports, private keys, or production webhook URLs.

## Disclaimer

These templates require configuration for your systems, data model, permissions, compliance requirements, and n8n version before production use.
